One-time setup: The public Cobalt API now requires authentication. You need a free Cobalt instance to use this. Here's the easiest way:
  1. Go to railway.app and sign in with GitHub (free)
  2. Click New Project → Deploy Template, search for cobalt
  3. Deploy it — Railway gives you a public URL like cobalt-xxx.up.railway.app
  4. In your Railway project → Variables, add: API_KEY_URL=file://keys.json and create a keys.json with a UUID key (or leave auth off)
  5. Paste your instance URL (and optional API key) below and click Save

Alternatively: Run cobalt locally with Docker:
docker run -p 9000:9000 -e API_URL=http://localhost:9000 ghcr.io/imputnet/cobalt
then use http://localhost:9000 as your instance URL.
